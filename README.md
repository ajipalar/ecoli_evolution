## Data Wrangling

This repository reproduces the _escherichia coli_ variant calling [workflow](https://datacarpentry.github.io/wrangling-genomics) in a reproducible manner using snakemake an Apptainer container.
Follows the tutorial at https://datacarpentry.github.io/wrangling-genomics

### Data
_E. coli_ evolution metadata was downloaded from https://datacarpentry.github.io/wrangling-genomics/files/Ecoli_metadata_composite.csv

The data consist of 25 generations.  rows and Z columns in tabular data.
Q citrate+ mutants have been recorded in Ara-3 strain. With Q hyper mutable mutants.
